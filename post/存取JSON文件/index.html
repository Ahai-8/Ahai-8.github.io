Godot Engine 学习笔记：存取JSON文件

· 2021-07-10 · # 游戏 # 开发 # Godot Engine

JSON文件常用于储存游戏存档和剧情文本。

常用方法与函数

  • file.get_as_text()：读取文件内容为纯文本
  • parse_json()：将参数解析为json
  • to_json()：将变量储存的内容转换为json格式
  • file.store_line()：将参数储存为单行文本

示例代码

extends Node

var data = {}

func load_game():
    var file = File.new()
    file.open("user://data.json",file.READ)
    var text = file.get_as_text()
    data = parse_json(text)
    file.close()


func save_game():
    var file
    file = File.new()
    file.open("user://data.json", File.WRITE)
    file.store_line(to_json(data))
    file.close()
